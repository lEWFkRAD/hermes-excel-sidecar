# Contributing

Search issues and PRs first; discuss large protocol/security changes before coding. Branch from `main`, keep one logical change per PR, use Conventional Commits, sign with `git commit -s`, and run the validation in `AGENTS.md`. Explain workbook, auth, installer, and rollback impact. Do not force-push after review starts.

AI-assisted contributions are welcome when disclosed, human-reviewed, and tested. Contributions are MIT licensed. Report vulnerabilities privately through GitHub.
